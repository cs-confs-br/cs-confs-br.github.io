About H5-Index

The H5-Index is a bibliometric metric that measures the impact and scientific productivity of academic conferences. An H5-Index of value h means that h articles published in the last 5 years have received at least h citations each.

This index is widely used to evaluate the quality and relevance of scientific conferences, providing an objective measure of publication impact.

The index is mainly calculated using Google Scholar [GS] metrics, but when unavailable, it is obtained from other sources like OpenAlex [OA], Crossref [CR] and many others.

Methodology

The data presented was collected and processed from recognized bibliographic databases, such as Google Scholar and OpenAlex. The analysis considers:

  • Publications from the last 5 years
  • Citations received by each article
  • Classification by knowledge area

Data is updated periodically to ensure information accuracy.

The search includes all 1400 conferences from CAPES CC 2022 Graduate Program evaluation (see CAPES document 09012022_CLASSIFICACAODEEVENTOSPARA20172020.xlsx), but since not all are listed under Google Scholar, we can add missing conferences manually using other bases per user request.

This project is open and collaborative, so if you miss some important conference just Open an Issue and we will try to help!

The data is available at website-2025.csv and also the processing logs generated from cs-confs-br-data project.
